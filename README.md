# Goldenage Enterprise 
**Building Bold Foundations Since 2007.**

A modern, fully responsive business website developed for **Goldenage Enterprise**, specializing in premium epoxy flooring, industrial coatings, and commercial space transformations. 

## 🛠️ Built With

This project is built using modern front-end web technologies to ensure a seamless, fast, and responsive user experience across all devices:

* **HTML5** - For semantic structural markup.
* **CSS3** - For custom styling and applying the brand's vibrant color palette.
* **JavaScript** - For interactive elements and smooth functionality.
* **Bootstrap** - For a robust, mobile-first responsive grid system and UI components.

## 🎨 Design & Color Palette

The website UI/UX is designed to reflect the brand's dynamic logo, utilizing a balanced mix of vibrant and neutral tones:
* **Primary Red:** `#D3212D`
* **Teal/Cyan:** `#009FBD`
* **Lime Green:** `#8CB83F`
* **Charcoal Gray:** `#4A4A4A` (Text)
* **Off-White:** `#F8F9FA` (Backgrounds)

## ✨ Key Features

* **Hero Banner:** Eye-catching high-resolution banner with strong call-to-actions.
* **Product Showcase:** Clean grid layouts highlighting various premium flooring solutions.
* **Responsive Design:** Flawless rendering on desktops, tablets, and mobile phones.
* **Cross-Browser Compatibility:** Optimized to work across all major modern browsers.
